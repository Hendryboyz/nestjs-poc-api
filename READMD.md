# Title

## Local Development

```bash
docker-compose up -d
cd api && npm run start:dev
```

## Techniques Stack

* Framework: NestJs
* Database
  * Mongo(NoSQL - document) for identity and users
  * PostgreSQL
  * Redis

### Tools

* Redis

  * [Medis(Mac Only)](https://getmedis.com/)
* Mongo
